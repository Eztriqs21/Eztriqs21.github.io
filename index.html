Supabase Setup

Connect your JEE HQ dashboard to Supabase for automatic cloud sync across all devices.

How to set up:
1. Create a project at supabase.com
2. Project Settings → API → copy URL & anon key
3. SQL Editor → create table jeehq_data (see block below)
4. Storage → New bucket → name exactly jeehq-assets → enable Public bucket
5. Storage → Policies → run the storage SQL block below
6. Paste settings here, save, then set a sync key in the sidebar
CREATE TABLE jeehq_data ( id uuid DEFAULT gen_random_uuid() PRIMARY KEY, sync_key text NOT NULL UNIQUE, data jsonb NOT NULL, updated_at timestamptz DEFAULT now() ); -- Storage policies — run ALL of this in SQL Editor (replace bucket name if needed) DROP POLICY IF EXISTS "jeehq_public_read" ON storage.objects; DROP POLICY IF EXISTS "jeehq_anon_upload" ON storage.objects; DROP POLICY IF EXISTS "jeehq_objects_select" ON storage.objects; DROP POLICY IF EXISTS "jeehq_objects_insert" ON storage.objects; DROP POLICY IF EXISTS "jeehq_objects_update" ON storage.objects; DROP POLICY IF EXISTS "jeehq_objects_delete" ON storage.objects; CREATE POLICY "jeehq_objects_select" ON storage.objects FOR SELECT USING (bucket_id = 'jeehq-assets'); CREATE POLICY "jeehq_objects_insert" ON storage.objects FOR INSERT WITH CHECK (bucket_id = 'jeehq-assets'); CREATE POLICY "jeehq_objects_update" ON storage.objects FOR UPDATE USING (bucket_id = 'jeehq-assets') WITH CHECK (bucket_id = 'jeehq-assets'); CREATE POLICY "jeehq_objects_delete" ON storage.objects FOR DELETE USING (bucket_id = 'jeehq-assets');
